# 🤖 Simple Rule-Based Chatbot (Python)

## 📌 Overview

This is a basic rule-based chatbot built using Python.
It responds to user inputs based on predefined conditions using simple programming concepts.

---

## 🚀 Features

* Responds to greetings like **hi, hello, hlo**
* Answers basic questions like **how are you**
* Handles unknown inputs with a default reply
* Exits the chat when the user types **bye**
* Handles empty input

---

## 🛠️ Concepts Used

* Functions (`def`)
* Loops (`while`)
* Conditional statements (`if-elif-else`)
* User input/output (`input()`, `print()`)

---

## ▶️ How to Run

1. Save the file as `task.py`
2. Open terminal or command prompt
3. Run the program:

```id="run123"
python task.py
```

4. Start chatting with the bot!

---

## 🧪 Sample Interaction

```id="sample12"
you: hello
bot: hi

you: how are you
bot: I'm fine, thanks!

you: hee
bot: I don't understand

you:
bot: please say something

you: bye
bot: Goodbye!
```

---

## ⚠️ Notes

* This chatbot is rule-based and does not use AI or machine learning
* It only responds to predefined inputs
* You can extend it by adding more conditions
