# 📈 Stock Portfolio Tracker

## 📌 Description
This is a simple Python project that calculates the total investment value of stocks entered by the user. It uses predefined stock prices and stores the result in a file.

---

## 🚀 Features
- User can enter stock name and quantity
- Supports multiple stock entries
- Calculates total investment automatically
- Saves result into a text file (`portfolio.txt`)
- Handles invalid stock inputs

---

## 🧠 Technologies Used
- Python

---

## 📂 Project Structure
CodeAlpha_Stock_Portfolio_Tracker/

│  
├── task2.py        # Main Python program  
├── portfolio.txt   # Stores total investment output  
├── README.md       # Project documentation  

---

## ▶️ How to Run
1. Open terminal  
2. Run the Python file:  
   python task2.py  
3. Enter stock names and quantities  
4. Type DONE to finish
